# calculator-mvc-project
This project is a calculator built with Python and Tkinter using the Model-View-Controller (MVC) architecture.

## Features
- Addition
- Subtraction
- Multiplication
- Division

## Technologies
- Python
- Tkinter
- MVC Design Pattern
